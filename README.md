# 🎵 听写小助手

一个帮助孩子轻松学习拼音、汉字、英语单词的微信小程序。

## ✨ 功能特性

- 🏷️ **拼音听写** - 智能拼音组合生成，按教学流程朗读
- 📝 **汉字听写** - 支持自定义词库
- 🌍 **英语单词** - 美式/英式发音可选
- 🏆 **成就系统** - 激励孩子学习
- 📋 **历史记录** - 查看学习轨迹

## 🚀 快速开始

### 1. 注册微信小程序

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册小程序账号
3. 获取你的 **AppID**

### 2. 注册百度 TTS

1. 访问 [百度智能云控制台](https://console.bce.baidu.com/)
2. 登录或注册百度账号
3. 在控制台搜索"语音合成"
4. 创建应用，获取 **API Key** 和 **Secret Key**
5. （可选）开通免费额度或付费套餐

### 3. 配置项目

1. 克隆或下载本项目
2. 复制 `config.example.js` 为 `config.js`，填入百度 TTS 的密钥：
   ```javascript
   module.exports = {
     baiduTTS: {
       apiKey: '你的百度 API Key',
       secretKey: '你的百度 Secret Key'
     }
   }
   ```
3. 修改 `project.config.json` 中的 `appid` 为你的小程序 AppID

### 4. 开通微信云开发

1. 在微信开发者工具中打开项目
2. 点击工具栏的 **云开发** 按钮
3. 开通云开发服务（基础版免费）
4. 创建环境（或使用免费环境）

### 5. 部署云函数

1. 在微信开发者工具左侧目录中找到 `cloudfunctions` 文件夹
2. 右键点击 `login` 文件夹 → **上传并部署：云端安装依赖**
3. 右键点击 `baiduTTS` 文件夹 → **上传并部署：云端安装依赖**
4. 等待部署完成

### 6. 创建云数据库

在云开发控制台 → 数据库 → 创建以下集合（权限建议设为"仅创建者可读写"）：

- `wordbooks` - 词库
- `dictation_history` - 听写历史
- `medals` - 成就奖牌
- `users` - 用户信息
- `feedback` - 反馈（可选）

### 7. 编译运行

在微信开发者工具中点击编译，项目即可运行！

## 📖 使用说明

### 拼音听写

1. 进入拼音页面，先选择声母
2. 再选择韵母
3. 系统自动生成有效音节
4. 可以开始听写，或保存到词库

### 汉字/英语听写

1. 可以直接使用预设词库
2. 也可以创建自定义词库
3. 设置语速、间隔、重复次数
4. 开始听写

### 个人中心

- 修改昵称和性别
- 查看听写统计
- 清除缓存
- 提交反馈

## 🔧 代码修改指南

### 修改拼音朗读格式

在 `pages/dictation/dictation.js` 中的 `splitPinyin()` 和 `playWordAudio()` 函数。

### 添加新的听写类型

1. 复制 `pages/pinyin/` 文件夹为新类型
2. 修改文件名和内容
3. 在 `app.json` 中注册新页面
4. 在首页添加入口

### 修改界面样式

所有页面的样式文件都在对应的 `.wxss` 文件中。使用 CSS 语法即可修改。

### 更换 TTS 服务

修改 `cloudfunctions/baiduTTS/index.js` 中的调用逻辑。

## 📁 项目结构

```
├── app.js                 # 主入口
├── app.json              # 小程序配置
├── config.js             # API 配置（需要自己填写）
├── config.example.js     # 配置示例
├── cloudfunctions/       # 云函数
│   ├── login/            # 登录云函数
│   └── baiduTTS/         # 百度语音合成
└── pages/                # 页面
    ├── index/            # 首页
    ├── pinyin/           # 拼音
    ├── chinese/          # 汉字
    ├── english/          # 英语
    ├── dictation/        # 听写中
    ├── wordbooks/        # 词库
    ├── history/          # 历史记录
    ├── medals/           # 成就
    └── profile/          # 个人中心
```
